FarmStore

Lightweight e‑commerce for farm inputs — demo & final year submission

Author: [Your Name]   •   Date: 2025-10-30

Key Features

  • Responsive product catalog with WebP/JPEG srcset
  • Client cart with localStorage persistence
  • Server-side checkout (Express + SQLite) with secure totals
  • Admin order management with realtime updates (Socket.IO)
  • Accessibility improvements (focus trap, ARIA, keyboard)

Architecture

Static frontend + Node.js backend. Image pipeline via Sharp generates multiple sizes and a manifest. Server emits events via Socket.IO for realtime updates.

Demo Script

  1. Start the server: npm start
  2. Open `farm.html`, browse products and add items to cart
  3. Checkout to create an order; watch admin panel receive `orders:new`
  4. Change order status in admin and observe realtime update
